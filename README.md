# 🎱 Bingo — Mostruário de Números Sorteados

Projeto desenvolvido com o objetivo de criar um painel visual para exibição de números sorteados em um bingo.

---

## 📌 Objetivo

Desenvolver uma interface simples e funcional capaz de:

- Exibir números sorteados
- Organizar visualmente os resultados
- Facilitar a visualização em tempo real
- Servir como apoio em eventos ou dinâmicas escolares

---

## 🚀 Funcionalidades

- Interface clara e objetiva
- Estrutura organizada em HTML
- Layout voltado para exibição pública
- Fácil adaptação para eventos

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS (estrutura visual)

---

## 💡 Aprendizado

- Organização de interface para uso prático
- Estruturação de layout simples e funcional
- Aplicação de HTML em solução de evento real
